# Nosok VPN — конкурсный frontend shell

Двухстраничный конкурсный визуал Nosok VPN: главная страница и личный кабинет.

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://127.0.0.1:3000`.

Для проверки проекта:

```bash
npm run typecheck
npm run lint
npm run test
npm run build
```

## Что готово

- тёплая «вязаная» дизайн-система и адаптивная главная;
- предварительный калькулятор стоимости;
- кабинет с разделами подписки, устройств, оплаты, профиля и помощи;
- выбор платформы, приложения и пошаговая инструкция подключения;
- мобильная нижняя навигация и доступная клавиатурная навигация.

## Граница текущего этапа

Это только визуальная оболочка. В проекте намеренно нет реальной авторизации, оплаты, выдачи или копирования VPN-ключа, QR-кода, базы данных, управления устройствами и истории операций. Кнопки в этих местах показывают готовую точку для будущей интеграции, не имитируя её результат.

Подробности: [контракт интеграции](docs/integration-contracts.md).
