# Иллюстрации Nosok VPN

20 самостоятельных WebP-иллюстраций для будущей замены плейсхолдеров «Место для авторской иллюстрации». Все сцены сделаны в одном тёплом вязаном мире, без текста, логотипов, QR-кодов и интерфейсов. В большинстве композиций действие находится справа, чтобы слева оставалось место под текст блока.

| Файл | Сцена и рекомендуемый блок |
| --- | --- |
| `01-hero-thread-connection.webp` | Маскот одной ниткой связывает телефон, ноутбук и ТВ. Главный hero о простом подключении. |
| `02-privacy-yarn-cocoon.webp` | Носок бережно укрывает телефон мягким коконом из пряжи. Блок о приватности. |
| `03-locations-felt-map.webp` | Маскот прокладывает нитяные маршруты по абстрактной карте из фетра. Раздел «Локации». |
| `04-subscription-calendar.webp` | Пряжа соединяет три пустые календарные карточки. Подписка и продление. |
| `05-connection-device-dock.webp` | Носок собирает простую связку из телефона, ноутбука и телевизора. «Как подключить». |
| `06-subscription-calendar-alt.webp` | Светлая альтернативная сцена с календарными карточками и пряжей. Вариант для калькулятора стоимости. |
| `07-locations-map-alt.webp` | Светлый вариант фетровой карты и маршрута. Альтернатива для карточки локаций. |
| `08-privacy-cocoon-alt.webp` | Вдумчивый носок упаковывает телефон в вязаную защиту. Альтернатива для приватности. |
| `09-hero-thread-alt.webp` | Альтернативный широкий сюжет про связанную технику. Вариант первого экрана. |
| `10-hero-thread-editorial.webp` | Более редакционный широкий вариант hero-сцены с нитями и устройствами. |
| `11-locations-route-editorial.webp` | Альтернативная сцена маршрутов для блока о серверах и странах. |
| `12-device-bundle.webp` | Носок бережно держит набор устройств на вязаных подушках. Тариф/несколько устройств. |
| `13-support-kindness.webp` | Маскот передаёт нить поддержки возле тёплой лампы. Блок помощи и поддержки. |
| `14-telegram-continuation.webp` | Бумажный самолётик движется по светящейся нити. Сообщение о доступе к Telegram и продлении. |
| `15-travel-ready.webp` | Носок собирает чемодан, а синяя нить уходит в пейзаж. VPN в поездках. |
| `16-gaming-smooth-route.webp` | Слегка суровый носок распутывает синюю нить у игрового экрана. Игровой маршрут. |
| `17-streaming-cozy.webp` | Маскот с попкорном у пустого экрана в уютной гостиной. Стриминг/развлечения. |
| `18-teamwork-thread.webp` | Два маскота вместе прокладывают нить между деревянными блоками. Совместная работа/надёжность. |
| `19-final-cta-paper-plane.webp` | Носок показывает бумажный самолётик и мягкий пьедестал. Финальный CTA. |
| `20-quiet-work.webp` | Спокойная работа за ноутбуком под вязаной аркой. Надёжное соединение для работы. |

Исходные PNG из генерации остаются вне проекта; здесь лежат только финальные оптимизированные WebP.
